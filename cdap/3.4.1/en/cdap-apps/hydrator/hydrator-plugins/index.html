ETL Plugins¶

Details of the required properties for sources, transformations (transforms), and sinks can be explored using RESTful APIs.

If you are creating a custom plugin to extend the existing system artifacts, its name should not collide with existing names for ease of use in the CDAP UI.

Shipped with CDAP, the plugins listed below (Hydrator Version 1.3.1) are available for creating ETL applications.
