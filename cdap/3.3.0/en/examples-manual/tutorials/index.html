Tutorials¶

  • WISE: Web Analytics
  • Netlens: Network Analytics
  • TwitterSentiment: Social Analytics
  • MovieRecommender: Recommender System

Designed to be completed in an hour or more, these tutorials provide deeper, in-context explorations of big data application development topics, leaving you ready to implement real-world solutions.

  • WISE: Web Analytics: Learn how to build a CDAP application to analyze a web site’s users behavior.
  • Netlens: Network Analytics: Learn how to build a CDAP application to analyze network packets to provide traffic stats and detect anomalies in the traffic patterns in real time.
  • TwitterSentiment: Social Analytics: Learn how build a CDAP application to perform sentiment analysis on Twitter data to understand consumer feelings and attitudes towards brands or topics in online conversations.
  • MovieRecommender: Recommender System: Learn how to build a CDAP application that uses collaborative filtering to provide movie recommendations for users.
