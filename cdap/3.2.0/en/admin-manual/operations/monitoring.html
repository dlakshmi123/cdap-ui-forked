Monitoring CDAP¶

Logs and Metrics¶

CDAP collects logs and metrics for all of its internal services. Being able to view these details can be really helpful in debugging CDAP applications as well as analyzing their performance. CDAP gives access to its logs, metrics, and other monitoring information through RESTful APIs as well as a Java Client.

See the Logging, Metrics, and Monitoring HTTP APIs, and the Java Client for more information.

Monitoring Utilities¶

CDAP can be monitored using external systems such as Nagios; a Nagios-style plugin is available for checking the status of CDAP applications, programs, and the CDAP instance itself.
