Designed to be completed in 15-30 minutes, these guides provide quick, hands-on instructions for building the “Hello World” of any development task with CDAP.

  • Analyzing CDAP Data from BI Tools: Learn how to integrate CDAP Datasets with Business Intelligence tools to perform data analysis.
  • Data Analysis with OLAP Cube: Use a Cube Dataset to perform online analytical processing of weblogs with CDAP.
  • Creating ETL Applications: Using CDAP System Artifacts to create ETL pipelines and applications.
  • Real-time Data Processing With a Flow: Learn how to process data in real time with CDAP. You will discover how easy it is to scale out applications on CDAP.
  • Ingesting Data into CDAP using Apache Flume: Learn how to use Flume to ingest data into CDAP in real time.
  • Consuming Data From Kafka: Learn how to consume data from a Kafka topic to process data in real time.
  • Batch Data Processing With CDAP: Learn how to run MapReduce programs to process data in CDAP.
  • Iterative Data Processing with Apache Spark: Learn how to run a Spark program to process data in CDAP.
  • Storing Timeseries Data: Learn how to store timeseries data in CDAP Datasets.
  • Consuming Twitter Data in Real-time: Learn how to consume a live tweets stream in real time, one of the common tasks of big data applications that power the social analytics.
  • Batch Data Processing with CDAP using Workflow: Learn how workflows can be used to execute MapReduce programs in parallel based on the evaluation of conditions.
