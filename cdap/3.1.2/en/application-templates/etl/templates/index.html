ETL Template and Plugins¶

Introduction

Details of templates and the required properties for sources, transformations, and sinks can be explored using RESTful APIs.

If you are creating a custom plugin to add to either the existing templates or your own template, its name needs to not collide with existing names.

Shipped with CDAP as part of the ETL Batch and ETL Realtime application templates, the plugins listed below are available for creating ETL adapters.

  • Sources
    • Batch
      • Database
      • File
      • KVTable
      • S3
      • Stream
      • Table
      • TPFSAvro
    • Real-time
      • AmazonSQS
      • DataGenerator
      • JMS
      • Kafka
      • Twitter
  • Transformations
    • LogParser
    • Projection
    • Script
    • ScriptFilter
    • StructuredRecordToGenericRecord
  • Sinks
    • Batch
      • Cube
      • Database
      • KVTable
      • Table
      • TPFSAvro
      • TPFSParquet
    • Real-time
      • Cube
      • Stream
      • Table

Exploring Application Template Details

Details on the available application templates can be obtained using the Application Template and Adapters HTTP RESTful API.
