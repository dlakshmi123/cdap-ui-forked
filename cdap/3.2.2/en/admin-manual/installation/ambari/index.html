This section is to help you install the Cask Data Application Platform (CDAP) on Hadoop systems that are HDP (Hortonworks Data Platform) clusters managed with Apache Ambari, the open source provisioning system for HDP.

For other distributions, managers, or formats:

  • For Cloudera Manager, follow these instructions.
  • For generic Apache Hadoop distributions, CDH clusters not managed with Cloudera Manager, or HDP clusters not managed with Ambari, follow these instructions.
  • For MapR, follow these instructions.
  • For the CDAP Standalone SDK, follow these instructions. (Developers’ Manual)

You can use Ambari to integrate CDAP into a Hadoop cluster by adding the CDAP Ambari Services to your Ambari server. Once you have restarted your server, you will able to use the Ambari UI (Ambari Dashboard) to install, start, and manage CDAP on Hadoop clusters.

These instructions cover the steps to integrate CDAP using Apache Ambari:

  • Adding the CDAP Service to Ambari: The prerequisite steps of preparing the package manager.
  • Adding CDAP to Your Cluster: The dependencies, both core and optional, required by CDAP.
  • Installing CDAP: Installing CDAP by running the Ambari Add Service Wizard, and starting CDAP.
  • Roadmap and Future Features: Additional features are planned for upcoming versions of the CDAP Ambari Service.
